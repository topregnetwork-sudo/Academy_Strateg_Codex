# Индекс проекта — Академия Стратег

## Как пользоваться этой папкой

Перед любой работой Codex обязан читать:

1. AGENTS.md
2. 00_START_HERE/00_PROJECT_INDEX.md
3. 01_CONTEXT/01_MAIN_GOAL.md
4. 02_STABLE_DATA/02_STABLE_CONTEXT.md
5. 07_CURRENT_TASKS/CURRENT_TASK.md

## Структура папок

00_START_HERE — карта проекта и стартовые инструкции.  
01_CONTEXT — главная цель, воронка, роли, архитектура.  
02_STABLE_DATA — подтверждённые стабильные данные проекта.  
03_N8N_WORKFLOWS — экспортированные n8n workflow JSON.  
04_GOOGLE_SHEETS — выгрузки таблиц и описания листов/колонок.  
05_PROMPTS — промпты для помощников, визуалов, техпомощника, Codex.  
06_REPORTS — отчёты этапов для переноса в новые чаты.  
07_CURRENT_TASKS — текущая задача и следующий шаг.  
08_ARCHIVE — старые версии, черновики, небоевые материалы.

## Главное правило

Не работать от отдельного файла.  
Работать от цели, воронки и стабильных данных.
