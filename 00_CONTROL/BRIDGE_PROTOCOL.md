# AS-CODEX-CHATGPT-LIVE-BRIDGE-001

## Назначение

Текущий чат используется как операционный терминал проекта «Академия Стратег».

## Формат обмена

- Отчёт Codex начинается с `[CODEX→CHATGPT]`.
- Задание, корректировка или стоп-сигнал архитектора начинается с `[CHATGPT→CODEX]`.
- После отчёта Codex ждёт ответ и не выполняет следующий опасный шаг самовольно.
- Пользователь не переносит сообщения между Codex и ChatGPT вручную.

## Формат отчёта

1. `[ЭТАП]`
2. `[ЧТО СДЕЛАНО]`
3. `[ЧТО ПОДТВЕРЖДЕНО]`
4. `[ЧТО ИЗМЕНЕНО]`
5. `[ЧТО НЕ ТРОГАЛИ]`
6. `[РИСКИ]`
7. `[НУЖНО РЕШЕНИЕ]`
8. `[ОДИН СЛЕДУЮЩИЙ ШАГ]`

## Секреты

В чат не передаются пароли, токены, API keys, credentials JSON, private keys и webhook secrets. Для обсуждения кода используются имя файла, функция, краткий diff, риск и план отката.

## Самостоятельно разрешено

Read-only аудит, локальный backup, анализ, подготовка diff и плана, локальные файлы и безопасные тесты без влияния на production.

## Требует отдельного подтверждения пользователя

Production Apps Script, боевые Google Sheets, массовая генерация, 3000 строк, n8n workflow, Telegram credentials, webhook URL, deploy Web App, публикация сайта и сообщения кандидатам.
